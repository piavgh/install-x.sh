# install-x.sh

Store all the shell scripts to install binaries easily

## Usage

For example, if you want to install Minikube:

- chmod +x install_minikube.sh
- ./install_minikube.sh

Now `minikube` binary is ready to be used at `/usr/local/bin/minikube`
